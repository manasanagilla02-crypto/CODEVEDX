Utility Usage Prediction Tool

A Machine Learning-based console application that predicts utility usage based on temperature and the number of users. This project demonstrates Python programming, CSV file handling, and basic Machine Learning using Linear Regression.

📌 Features
Menu-driven console application
Add and manage utility usage data
CSV file handling using Pandas
Linear Regression model for prediction
Utility usage prediction based on input values
Modular code structure
Exception handling
Beginner-friendly Machine Learning project
