# AI Based Fake News Detection Tool

## Description

A Machine Learning and Natural Language Processing (NLP) project that classifies news articles as **REAL** or **FAKE** using text preprocessing and TF-IDF vectorization.

## Features

* Fake/Real news classification
* News text input
* Text preprocessing
* TF-IDF vectorization
* Confidence score prediction
* Trained ML model storage

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NLTK
* TF-IDF Vectorizer
* Pickle
