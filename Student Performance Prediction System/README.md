 Student Performance Prediction System

 Description

A Machine Learning project that predicts a student's final performance based on attendance, study hours, and internal marks. The project demonstrates the complete ML workflow, including data preprocessing, feature selection, model training, prediction, and performance evaluation.

 Features

* Student performance prediction
* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection
* Model accuracy evaluation
* User input-based prediction
* Data visualization using charts

 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
